# 🚀 Enable GitHub Pages - Quick Setup

Your E-Commerce Backend documentation is now ready! The `index.html` has been pushed to the repository root. 

Follow these **3 simple steps** to go live:

---

## Step 1️⃣ Open Repository Settings

Go to: **https://github.com/abhishek2k21/E-Commerce-Backend/settings**

(Or: Repository → Settings tab)

---

## Step 2️⃣ Navigate to Pages

In the left sidebar, find and click **"Pages"** (under "Code and automation" section)

---

## Step 3️⃣ Configure GitHub Pages

In the **"Build and deployment"** section:

1. **Source** dropdown: Select `Deploy from a branch`
2. **Branch** dropdown: Select `main`
3. **Folder** dropdown: Select `/ (root)`
4. Click **"Save"** button

---

## ⏳ Wait for Deployment

GitHub will now build your site. This takes **1-2 minutes**.

You'll see a green checkmark ✅ when complete.

---

## ✨ Your Live Site

Once deployed, your documentation will be available at:

### 🌐 **https://abhishek2k21.github.io/E-Commerce-Backend/**

---

## ✅ Verify It's Working

1. Wait 2 minutes after saving
2. Visit: https://abhishek2k21.github.io/E-Commerce-Backend/
3. You should see your beautiful E-Commerce API documentation site!
